Contenido

En este tema aprendí sobre los modelos de transporte y asignación, que básicamente sirven para encontrar la mejor forma de distribuir recursos de un lugar a otro, buscando obtener el mejor resultado posible. Dependiendo del problema, el objetivo puede ser minimizar costos, tiempos o distancias, o también maximizar beneficios o el impacto que genera una determinada distribución.

También vimos que existen modelos balanceados, cuando la oferta total es igual a la demanda total, y desbalanceados, cuando estas cantidades no coinciden. En esos casos es necesario agregar una fila o columna ficticia para poder resolver correctamente el problema.

Otro concepto importante fue el modelo de asignación, que es un caso especial del modelo de transporte. Aquí cada origen solo puede asignarse a un único destino y viceversa, como cuando se asigna un trabajador a una máquina o un camión a una ruta específica.

Para resolver este tipo de problemas seguimos un proceso de dos etapas. Primero utilizamos el método de Aproximación de Vogel, que permite obtener una muy buena solución inicial utilizando penalizaciones para decidir las mejores asignaciones. Después aplicamos el método del Salto de Piedra, que sirve para comprobar si esa solución ya es la óptima o si todavía existe una forma de mejorarla.

Método de Aproximación de Vogel

Objetivo: obtener una solución inicial de alta calidad usando penalizaciones

¿Qué es una penalización?

Es la diferencia entre los dos costos de oportunidad más bajos de cada fila o columna. Indica cuánto "perdemos" si no elegimos la mejor opción. Siempre se asigna en la fila/columna de mayor penalización, en la celda de menor costo de oportunidad.

A

Convertir a Costo de Oportunidad

Como es maximización, convertimos usando: Costo de oportunidad = 16 (mayor ganancia) − ganancia de la celda. Así el costo más bajo = mayor ganancia original.

Método del Salto de Piedra

Objetivo: verificar si la solución de Vogel puede mejorar

¿Cómo funciona?

Para cada celda vacía, formamos un "circuito cerrado": empezamos en la celda vacía (+), saltamos solo por celdas ocupadas alternando entre filas y columnas, alternando los signos + y −, hasta volver al inicio.

Calculamos Δ = suma de ganancias con esos signos. Si algún Δ > 0 conviene reasignar (mejora la ganancia). Si todos Δ ≤ 0, la solución es óptima.

Fórmula: m + n − 1 = celdas ocupadas
3 orígenes + 4 destinos − 1 = 6
Celdas ocupadas en Vogel: X₁₁, X₁₄, X₂₂, X₂₃, X₂₄, X₃₂ = 6 ✓

Modelo de Asignación — Método Húngaro

Objetivo: asignar cada camión a exactamente una ruta maximizando el rendimiento total

¿Qué es el Método Húngaro?

Es una variante del modelo de transporte donde cada recurso (camión) se asigna a exactamente una actividad (ruta). Se convierte a costo de oportunidad, se reducen filas y columnas, y se busca una asignación donde cada fila y columna tenga exactamente un cero independiente.
