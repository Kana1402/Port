Control Estadístico

Ejercicio: Control Estadístico del Proceso de Corte de Precisión de Alambre

Contenido

Esta presentación del curso de Control Estadístico se enfoca en algo clave en la industria: dejar de solo revisar el producto final y empezar a controlar el proceso mientras ocurre, para evitar errores antes de que se conviertan en pérdidas.

La calidad se entiende como el cumplimiento de estándares y la reducción de defectos para satisfacer al cliente. Para eso se usa el Control Estadístico de la Calidad (CEC), que se basa en datos y gráficas para monitorear el comportamiento del proceso en tiempo real.

Un concepto central es la variabilidad, que puede venir de causas comunes (variación normal del proceso) o causas especiales (problemas puntuales que sí deben corregirse). Para detectarlas se usan gráficas de control con límites estadísticos que indican cuándo un proceso está bajo control o no.

Dependiendo del tipo de datos, se aplican gráficas para variables (como peso o tiempo) y para atributos (como defectos o piezas fallidas), lo que permite tomar decisiones más informadas dentro de la gestión del proceso.

En el ejercicio práctico, se analiza una empresa de corte de precisión de alambre durante 24 horas, tomando muestras de 4 piezas por hora. El objetivo es verificar si el proceso se encuentra bajo control estadístico.

Ejercicio detallado

Control Estadístico del Proceso de Corte de Precisión de Alambre

🤔 ¿Por qué hacemos este control?

En una fábrica de corte de alambre, cada pieza debería tener la misma longitud. Pero en la práctica, siempre hay pequeñas variaciones: la máquina se calienta, la materia prima cambia levemente, el operario ajusta la tensión, etc.

La pregunta clave es: ¿esas variaciones son normales o hay algo que está fallando? Si la variación es demasiado grande o el promedio empieza a desviarse del centro, significa que algo salió mal y hay que corregirlo. El Control Estadístico nos da límites matemáticos precisos para responder esa pregunta.

Enunciado

En una empresa dedicada al corte de precisión de alambre se desea verificar si el proceso se encuentra bajo control estadístico. Durante 24 horas consecutivas se tomaron muestras de 4 piezas por hora. Para cada muestra se calculó el promedio (X̄) y el rango (R). Con la información obtenida se solicita construir los gráficos X̄ y R, calcular sus límites de control y determinar si el proceso está bajo control estadístico.

Solución paso a paso — Explicado

1

Identificar la información y las constantes estadísticas

📋 ¿Por qué usamos constantes (A₂, D₃, D₄)?

Los límites de control de una gráfica X̄-R no se calculan con la desviación estándar directamente (eso sería complicado). En cambio, los estadísticos desarrollaron unas constantes tabuladas que dependen únicamente del tamaño de muestra (n).

Estas constantes ya tienen integrado el factor estadístico de ±3 desviaciones estándar (que cubre el 99.73% de la variación normal). Solo necesitamos multiplicarlas por el rango promedio para obtener los límites.

Datos del problema:

  • • k = 24 muestras (horas)
  • • n = 4 piezas por muestra
  • • Se conoce X̄ y R de cada hora

Constantes para n = 4:

  • • A₂ = 0.729 → para gráfica X̄
  • • D₃ = 0 → LCI del gráfico R
  • • D₄ = 2.282 → LCS del gráfico R

Se obtienen de la tabla de constantes de control estadístico.

2

Calcular la Gran Media (X̄̄)

📋 ¿Qué es la Gran Media y cómo se calcula?

La Gran Media (X̄̄, pronunciado "X doble barra") es el promedio de todos los promedios. Representa el centro del proceso — el valor alrededor del cual debería oscilar todo.

Se calcula sumando todos los 24 promedios (X̄ de cada hora) y dividiendo entre 24. Será la línea central (LC) de la gráfica X̄.

// Sumamos los 24 promedios X̄ de cada hora
ΣX̄ = 3.25 + 3.11 + 3.22 + ... + 2.94 + 2.64
ΣX̄ = 71.32

// Dividimos entre el número de muestras (k = 24)
X̄̄ = 71.32 ÷ 24

X̄̄ = 2.972  ← Esta es la Línea Central de la gráfica X̄

✅ ¿Qué significa 2.972?

Significa que en promedio general, el alambre cortado mide 2.972 unidades. Si el proceso estuviera perfectamente centrado, todos los promedios deberían oscilar cerca de este valor. Los límites se construirán alrededor de este centro.

3

Calcular el Rango Promedio (R̄)

📋 ¿Qué es el Rango Promedio y por qué lo necesitamos?

El rango (R) de una muestra = valor máximo − valor mínimo de las 4 piezas de esa hora. Mide la variabilidad dentro de cada muestra.

El Rango Promedio (R̄) es el promedio de todos los rangos. Lo necesitamos porque las fórmulas de los límites de control de ambas gráficas (X̄ y R) dependen de R̄. Es la manera de estimar la variación natural del proceso.

// Sumamos los 24 rangos R de cada hora
ΣR = 0.71 + 1.18 + 1.43 + ... + 1.58 + 0.97
ΣR = 24.37

// Dividimos entre el número de muestras (k = 24)
R̄ = 24.37 ÷ 24

R̄ = 1.015  ← Esta es la Línea Central de la gráfica R
4

Calcular los límites del gráfico X̄ (Promedios)

📋 ¿Cómo se calculan y qué significan los límites?

Los límites del gráfico X̄ definen el rango dentro del cual deberían caer los promedios de cada hora si el proceso está bajo control.

  • LCS (Límite de Control Superior) = X̄̄ + A₂ × R̄ → la "barrera superior"
  • LC (Línea Central) = X̄̄ → el centro esperado del proceso
  • LCI (Límite de Control Inferior) = X̄̄ − A₂ × R̄ → la "barrera inferior"

Si algún promedio cae fuera de estas barreras → 🚨 señal de alarma: algo anormal ocurrió en ese momento.

// Fórmula: LCS = X̄̄ + (A₂ × R̄)
LCS = 2.972 + (0.729 × 1.015)
LCS = 2.972 + 0.740
LCS = 3.712  ← Todo X̄ debe estar BAJO este valor

LC  = 2.972  ← Centro del proceso

// Fórmula: LCI = X̄̄ - (A₂ × R̄)
LCI = 2.972 − (0.729 × 1.015)
LCI = 2.972 − 0.740
LCI = 2.231  ← Todo X̄ debe estar SOBRE este valor

5

Calcular los límites del gráfico R (Rangos)

📋 ¿Para qué sirve el gráfico R y cómo se calcula?

Mientras el gráfico X̄ controla el centro del proceso, el gráfico R controla la variabilidad. Aunque el promedio esté bien, si los rangos son muy grandes significa que las piezas de una misma muestra son muy diferentes entre sí (proceso inestable).

  • LCS = D₄ × R̄ → si un rango supera este valor, hubo variabilidad excesiva en esa hora.
  • LCI = D₃ × R̄ → para n=4, D₃ = 0, entonces LCI = 0. No puede haber rango negativo.
// Fórmula: LCS_R = D₄ × R̄  (D₄ = 2.282 para n=4)
LCS = 2.282 × 1.015
LCS = 2.317  ← Todo R debe estar BAJO este valor

LC  = 1.015  ← Rango promedio esperado

// Fórmula: LCI_R = D₃ × R̄  (D₃ = 0 para n=4)
LCI = 0 × 1.015
LCI = 0      ← No existe límite inferior real (rango ≥ 0 siempre)

6

Verificar si el proceso está bajo control

📋 ¿Cómo se verifica y qué significa "bajo control"?

Comparamos cada uno de los 24 valores de X̄ contra los límites del gráfico X̄, y cada uno de los 24 valores de R contra los límites del gráfico R.

Regla: Si NINGÚN punto supera los límites → el proceso está bajo control estadístico (solo hay variación normal). Si ALGÚN punto supera los límites → hay una causa especial que debe investigarse.

Gráfica X̄ — Verificación:

LCS (máximo permitido) 3.712
Mayor X̄ observada (Hora 4) 3.39 ✅
LCI (mínimo permitido) 2.231
Menor X̄ observada (Hora 24) 2.64 ✅

→ Todos los 24 promedios están dentro del rango [2.231 – 3.712] ✅

Gráfica R — Verificación:

LCI (mínimo permitido) 0

Conclusión Final

El proceso SÍ está bajo control estadístico ✅

📌 Interpretación detallada:

Al comparar todos los promedios (X̄) y rangos (R) con sus respectivos límites de control, ningún valor supera los límites establecidos. Esto nos dice dos cosas importantes:

  • El proceso está centrado alrededor de 2.972 y no presenta tendencias hacia arriba o abajo.
  • La variabilidad es normal: los rangos oscilan sin patrones anómalos, dentro de lo esperado para un proceso estable.

Las variaciones observadas son propias del funcionamiento normal del equipo (causas comunes). No se detectaron causas especiales (como un operario distinto, una materia prima diferente, o una falla mecánica) que requieran intervención.

➡️ Recomendación gerencial: El proceso puede continuar sin ajustes. Sin embargo, se sugiere implementar una revisión preventiva periódica del equipo de corte para mantener este nivel de estabilidad a largo plazo.
