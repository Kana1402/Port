Tema 6

Control Estadístico

Contenido

El Control Estadístico de Procesos (CEP) es una herramienta que ayuda a verificar si un proceso está funcionando de forma estable o si algo se salió de lo normal. En lugar de tomar decisiones por intuición, se analizan datos reales para detectar cambios, prevenir errores y mantener una calidad constante en la producción.

Una de las herramientas más utilizadas son las gráficas de control X̄ y R. La gráfica X̄ permite observar cómo cambia el promedio de las muestras a lo largo del tiempo, mientras que la gráfica R muestra qué tanta variación existe entre los datos de cada muestra. Si todos los puntos permanecen dentro de los límites de control, significa que el proceso está trabajando de manera estable; si aparecen puntos fuera de esos límites o patrones poco comunes, es una señal de que algo debe revisarse antes de que afecte la calidad del producto.

Ejercicio detallado

Control Estadístico del Proceso de Corte de Precisión de Alambre

Enunciado

En una empresa dedicada al corte de precisión de alambre se desea verificar si el proceso se encuentra bajo control estadístico. Durante 24 horas consecutivas se tomaron muestras de 4 piezas por hora. Para cada muestra se calculó el promedio (X̄) y el rango (R). Con la información obtenida se solicita construir los gráficos X̄ y R, calcular sus límites de control y determinar si el proceso está bajo control estadístico.

Solución paso a paso explicada

1

Identificar la información del problema

Antes de comenzar los cálculos debemos identificar los datos que nos proporciona el ejercicio.

  • 24 muestras tomadas durante 24 horas.
  • n = 4 piezas por muestra.
  • Se conoce el promedio (X̄) y el rango (R) de cada muestra.

Como el tamaño de muestra es 4, utilizaremos las constantes estadísticas: A₂ = 0.729, D₃ = 0 y D₄ = 2.282.

2

Calcular la Gran Media (X̄̄)

La Gran Media representa el promedio general de todo el proceso. Para obtenerla sumamos todos los promedios y dividimos entre las 24 muestras.

ΣX̄ = 71.32

X̄̄ = 71.32 / 24

X̄̄ = 2.972
3

Calcular el Rango Promedio

El rango promedio indica la variación promedio entre las piezas de cada muestra.

ΣR = 24.37

R̄ = 24.37 / 24

R̄ = 1.015
4

Calcular los límites del gráfico X̄

Ahora utilizamos la constante A₂ para construir los límites de control.

LCS = X̄̄ + A₂R̄
LCS = 2.972 + (0.729 × 1.015)

LCS = 3.712

LC = 2.972

LCI = 2.972 − (0.729 × 1.015)

LCI = 2.231
5

Calcular los límites del gráfico R

Con las constantes D₃ y D₄ calculamos los límites del gráfico de rangos.

LCS = D₄ × R̄
LCS = 2.282 × 1.015

LCS = 2.317

LC = 1.015

LCI = 0 × 1.015

LCI = 0
6

Verificar si el proceso está bajo control

Finalmente se comparan todos los valores de X̄ y R con sus respectivos límites de control.

Resultado

  • El menor promedio observado fue 2.64.
  • El mayor promedio observado fue 3.39.
  • Todos los promedios se encuentran entre 2.231 y 3.712.
  • Todos los rangos también permanecen entre 0 y 2.317.

Como ningún punto supera los límites de control, el proceso se considera estable.

Conclusión

El proceso SÍ está bajo control estadístico ✅

Interpretación

Al comparar todos los promedios (X̄) y todos los rangos (R) con sus respectivos límites de control, se observa que ningún valor sobrepasa los límites establecidos. Esto significa que las variaciones observadas son propias del funcionamiento normal del proceso y no existen evidencias de causas especiales que afecten la producción. En consecuencia, el proceso de corte de precisión de alambre puede considerarse estable y bajo control estadístico.
